# Стандартный алгоритм умножения матриц

## Условия умножения
Матрицу **A** размером `m × n` можно умножить на матрицу **B** размером `n × p`.  
Результатом будет матрица **C** размером `m × p`.

## Математическая формула
Для каждого элемента `C[i][j]` результирующей матрицы:


C[i][j] = Σ (A[i][k] × B[k][j]) для k = 0..n-1


## Алгоритм (псевдокод)
```bash
for i = 0 to m-1:  
    for j = 0 to p-1:  
        C[i][j] = 0  
        for k = 0 to n-1:  
            C[i][j] += A[i][k] × B[k][j]  
```

## Пример:

A = | 1  2  3 |    B = | 7  8  |
    | 4  5  6 |        | 9  10 |
                       | 11 12 |


C[0][0] = A[0][0]×B[0][0] + A[0][1]×B[1][0] + A[0][2]×B[2][0]
        = 1×7 + 2×9 + 3×11 = 7 + 18 + 33 = 58

C = | 58  64 |
    | 139 154 |


## Сложность алгоритма
- **Временная сложность**: O(m × n × p) = O(n³) для квадратных матриц
- **Пространственная сложность**: O(m × p) для хранения результата
